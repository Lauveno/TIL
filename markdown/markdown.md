## 제목

'#'의 개수에 따라 제목의 중요도가 달라져요.

## 리스트

### 순서 있는 리스트

1. 식용유를 두른다
2. 파를 넣고 파기름을 낸다
3. 계란을 풀어서 익힌다
4. 간장을 넣는다
5. 재료를 넣어 볶는다
6. 밥을 넣어 볶는다
7. 볶음밥 완성



### 순서 없는 리스트(-)

- cli 학습
  - 파일
    - `touch`
    - `rm`
    - `start`
  - 폴더
    - `mkdir1`
    - `rm -r`
    - `cd`
- git 복습
- md 작성 익숙해지기



## 인라인 코드(`)

파일을 생성하기 위해서 `touch` 명령어를 사용한다

`cd`를 통해 폴더간 이동을 한다
